####Add the speech recognition plugin
Speech recognition plugin written for Android can be installed like this:

First add platform : `cordova platform add android`

Second, add plugin: `cordova plugin add https://github.com/poiuytrez/SpeechRecognizer`

More information on how to use the plugin and example code please following [this](https://github.com/ajithnn/GVTech/tree/master/VoiceToText/plugins/com.phonegap.plugins.speech) link.

####Build app

`$ cordova build`
